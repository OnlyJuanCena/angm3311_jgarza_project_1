### Maya Building Generator

GitHub URL:
https://github.com/OnlyJuanCena/angm3311_jgarza_project_1

Challange Features:
- Created procedural windows that properly scale to any building's size or dimensions.
- Added a Random Windows modifier which allows buildings to have a random amount of procedurally generated windows.
- Added a Delete and Delete All button to allow for easy tweaking and regeneration of procedural buildings.
